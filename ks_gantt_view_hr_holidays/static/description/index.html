KEY HIGHLIGHTS OF

The Gantt View Time Off

Leave management becomes easy and convenient with Gantt View Time Off!

Action Bar

The Action Bar contains different tools that include:

Duration

Once you click on the duration button, it displays the number of leaves applied by each employee in the Gantt View Time Off.

Grid

By using the grid feature, you can hide the left grid panel that contains the title, duration, start time, and owner. It allows you to view the complete leave data in the Gantt View Time Off.

Resource Panel

The Resource Panel shows the total number of leaves applied by each employee along with working hours and allocated tasks. With this feature, you can manage the employee's leaves while checking out how many working hours and tasks will be affected during their leaves.

Zoom to Fit

Zoom to Fit feature gives an easily readable Gantt View of large data according to the user's screen.

Critical Path

This feature works on the current view only. It shows the dependency of the current leave on the previous one.

Days Off

The term "day off" is used to highlight the week off and weekends in the Gantt View.

Today

Today's feature will scroll you to the current date in the timeline and highlight it.

Scaling

With this feature, you can set the duration such as day, week, month, year, and more. to display the leave information in different patterns. In this way, you can make your data more readable in the Gantt View Time Off as per your convenience.

Sorting

This feature allows you to sort your leave data in different orders, like Name Ascending, Name Descending, New to Old, and more.

Full Screen Mode

If you have a small screen, then you can use this feature to get a full-screen mode of the Gantt View.

Gantt View Grid

Give freedom to customize your view!

Left Grid

The left grid will display leave data with the leave title, duration, start time, and owner.

Title

The type of leaves you applied

Duration

Time duration of the leaves

Start Time

When the leaves start and end

Owner

Who applied for the leaves?

Add Button

Once you click the (+) icon, a default Odoo form will appear on the screen where you can apply for your leave and save it.

Edit button

The edit option allows you to modify your applied leave.

Delete Button

It enables you to delete your applied leave.

Right Grid
Tooltip

When you hover a mouse over a leave-bar in the Gantt View, the tooltip feature will show the leave details including Title, Duration,Start date, End date and Stage.

Quick Info

When you click on the leave-bar in the Gantt view, a quick info will appear that displays the Start date, End date of the leave along with a title. Also, it gives an option to delete and edit the leave.

Export

With this feature you can export your employee's leave report in different formats like Excel, PDF, PNG, Json, XML, Primvera P6 and iCal.

Gantt Settings

Different configuration to customize your Gantt View

Tooltip Settings

You can modify the leave information display in the Gantt View Time Off by enabling or disabling the tooltip setting options like duration, start date, end date, and more.

Gantt Chart Settings

Different configuration to customize your Gantt View

Dynamic Text

Generally, leave descriptions are too long, which doesn't fit into the Gantt view. When you enable "Dynamic Text," the leave info will be automatically adjusted and displayed in the Gantt View.

Quick Info

This feature gives an option to delete or edit the leave when you click on it in the Gantt View Time Off. You can enable or disable this feature as per your choice.

Stage Color

You have the option to highlight the leave status in different colors, like to submit, to approval, approved, and more. This will provide a better view of the leaves in the Gantt View and make it easy to understand.

What is Gantt View Time Off ?

Gantt View Time Off module is the one-stop platform to manage the leaves of employees for an organization/ business.

How will it help in leave management?

With it’s easy-to-use functionalities, anyone can leverage the Gantt Chart to organize regular leave routines. There are various leave types that can be set up along with specific durations, extra notes, etc. Critical path and leave dependencies can also be set up easily.

Is this app compatible with Odoo Enterprise?

No, it is not currently compatible with the Odoo Enterprise. However, future updates can make it compatible with Odoo Enterprise later.

How to create time off request?

These are the steps to create the time off request:
1. Click on the create/plus icon button on the Gantt view,
2. Select the leave type,
3. Add leave start/end date,
4. Add description(regarding the leave) and then save it.

Is it possible to modify my leaves?

Yes, it is possible to modify the leaves by editing a saved leave.

Who can manage the leaves of other employees?

The admin/manager can manage the leaves of other employees easily.

In which formats can the Gantt View Time Off report be exported?

The Gantt View Time Off report can be exported in many formats like Excel, PDF, PNG, JSON, MS Project(XML), Primavera P6, and iCal.
