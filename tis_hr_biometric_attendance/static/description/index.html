Biometric Attendance

This module helps to integrate Odoo HR Attendance with ZK attendance machine.

Configure the device with Odoo (Attendances >> Biometric Device Config)

Configure a static IP, Port and Timezone for the system. If the biometric device has password, you can configure it by enabling 'Is Password Set' field. After configuration click the 'Test Connection' button.

If your connection success, server shows a success message

If your connection fails, server shows an error message

Synchronize employees with biometric device

Click the 'Sync Employees' button to synchronize employees with biometric device.

If employees synced with biometric device, you can see a popup with success message.

You can see three menus in employee form.
1. Scan with biometric - Click to scan employee with biometric device
2. Update with biometric - Click to update employee details with biometric device
3. Remove from biometric - Click to remove employee details from biometric device

Click Scan to enroll employee with biometric device.

Click 'Update' to update employee details with biometric device.

Click 'Remove 'to remove employee details from biometric device.

After updating the employee details, a line will create under the 'Biometric Devices' tab. This line maps the same ID present on the biometric device to the Biometric User Id field.

Download Attendance Logs

Click the 'Download Attendance' button to download attendance log.

After downloading attendance log server shows a success message.

Under the Manager menu, you can see three new menus.

1. Biometric Attendance Log - shows the downloaded attendance log of employees
2. Calculate Attendance - calculate the attendance and shows the difference in attendances menu.
3. Attendance Report - excel report of attendance

Biometric attendance log tree view

To calculate attendance click 'Calculate Attendance' menu. Then click on the Calculate button.

Select 'Attendances' from the Manager menu to see the employees attendances.

By clicking 'Attendance Report' menu, you can see a popup. Select the report type , From date and To date from here.
