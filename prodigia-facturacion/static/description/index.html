Módulo de Facturación de Prodigia

El módulo agrega como PAC a Prodigia y añade un campo dentro de las configuraciones de facturación para indidar el contrato del cliente para consumir los servicios de facturación que tiene publicados Prodigia.

Una vez capturados los datos de facturación la base esta cofnfigurada correctamente y odoo podrá emitir CFDI mediante Prodigia.
